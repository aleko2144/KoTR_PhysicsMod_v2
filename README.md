# KoTR_PhysicsMod_v2
King of the Road / Hard Truck 2 roadtrain movement plugin.

Функция обработки движения автопоездов, вынесенная в asi-плагин. Сцепление/расцепление полуприцепа с тягачом работает некорректно. Для сцепки нужно нажать кнопку <T> (или другую, выбранную в настройках игры), а затем вызвать 911 (или восстановить грузовик при помощи читов). Либо же пересобрать плагин, увеличив значение параметра couplerAttachDist. Расцепление не работает. В остальном всё соответствует оригинальной игре.
